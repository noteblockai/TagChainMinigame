# 🎮 마인크래프트 꼬리잡기 미니게임 플러그인

Spigot 1.21.x 기반으로 제작된 **순환 추적형 PvP 미니게임 플러그인**입니다.
각 플레이어는 단 한 명의 타겟을 가지며, 타겟을 잡으면 다음 타겟을 이어받는 방식으로 게임이 진행됩니다.

플러그인 다운로드는 우리들의 마인크래프트 공간에서 다운받아가실수 있습니다

[만약에 이분이 다른 방송에서 합방목적으로 이 컨텐츠 플러그인을 진행할경우 그 담당 BJ에게 저작권으로 신고넣겠습니다]







## 📌 게임 개요

플레이어들은 서로를 추적하는 **원형 구조**를 형성합니다.

```
A → B → C → D → A
```

* 각 플레이어는 **자신의 타겟 1명만 존재**
* 타겟을 잡으면 → **그 타겟의 목표를 이어받음**
* 마지막 **2명이 남으면 승리**

---

## ⚙️ 게임 진행 방식

### 🟡 1단계: 준비 단계 (2분)

* `/game start` 입력 시 시작
* 모든 플레이어가 게임 위치로 이동
* 타겟이 랜덤으로 설정됨
* **2분 동안 도망 가능**
* ❌ 공격 불가
* ❌ Glow 표시 없음

---

### 🔴 2단계: 게임 진행

* 2분 후 자동 시작
* 이후 반복:

```
1분 대기 → 3초 동안 타겟 Glow 표시 → Glow 해제 → 반복
```

### 🔦 Glow 시스템

* 자신의 타겟만 Glow로 보임
* 다른 플레이어는 절대 볼 수 없음
* UI 표시 없음 (완전 숨김)

❌ 금지된 UI:

* BossBar
* ActionBar
* Title
* 채팅 안내

👉 오직 Glow로만 타겟 확인 가능

---

## ⚔️ 잡기(탈락) 시스템

* 타겟을 공격하면 즉시 탈락 처리
* 실제로 죽지 않음 (데미지 취소됨)

### 탈락 시 처리

* 타겟 체인에서 제거
* 관전자 모드로 전환
* 로비로 이동

```
A → B → C → D → A

B 탈락 →

A → C → D → A
```

---

## 🏆 승리 조건

* **2명 남으면 게임 종료**
* 남은 2명 모두 승리

---

## 🚪 접속/이탈 처리

### 준비시간(2분) 동안

* 나가면 즉시 탈락
* 메시지 출력:

```
[플레이어]님이 접속하지 않아 탈락했습니다.
```

* 2분 내 재접속 안 하면 탈락

---

### 게임 진행 중

* 나가면 즉시 탈락
* 체인 자동 재연결

---

## 👁 관전자 시스템

### `/game spectator <플레이어>`

해당 플레이어를 관전자로 전환

### 관전자 특징

* 타겟 체인 제외
* 공격 불가 / 피격 불가
* Glow 보이지 않음
* 게임 승리 대상 제외
* 자유롭게 관전 가능

---

## 🎮 명령어 목록

모든 명령어는 **OP 전용**

### ▶ 게임 시작

```
/game start
```

* 게임 시작
* 2분 준비시간 후 진행

---

### ⛔ 게임 중단

```
/game stop
```

* 게임 강제 종료
* 모든 효과 제거
* 전원 로비 이동

---

### 🏠 로비 설정

```
/game setlobby
```

* 현재 위치를 로비로 설정

---

### 🎯 게임 위치 설정

```
/game setgame
```

* 현재 위치를 게임 시작 위치로 설정

---

### 👁 관전자 지정

```
/game spectator <플레이어>
```

* 해당 플레이어를 관전자 모드로 전환
